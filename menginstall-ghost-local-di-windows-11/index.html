Panduan ini membimbing kamu menjalankan Ghost secara lokal di Windows 11, lengkap dengan solusi error umum.

üìã Prasyarat

  • Windows 10/11 (64-bit)
  • Node.js v22.x (non-LTS; perlu build manual)
  • NPM (terinstal bersama Node.js)
  • Visual Studio Build Tools dengan workload Desktop development with C++
  • Akses Administrator untuk terminal & file hosts

üõ†Ô∏è 1. Instal Ghost CLI

Buka PowerShell atau Command Prompt sebagai Administrator, lalu jalankan:

npm install -g ghost-cli

üóÇÔ∏è 2. Siapkan Folder Instalasi

Pindah ke drive D: dan buat direktori ghost:

D:
mkdir ghost
cd ghost

üöÄ 3. Install Ghost Lokal

Jalankan perintah:

ghost install local

Ghost akan:

  • Mengecek environment
  • Mengunduh Ghost versi terbaru
  • Menggunakan SQLite sebagai database
  • Menjalankan server di http://localhost:2368

üîß 4. Mengatasi Error ‚ÄúCannot find module 'sqlite3'‚Äù

Jika muncul error tersebut:

  1. Pastikan Visual Studio Build Tools sudah terpasang dengan workload Desktop development with C++.
  2. Instal sqlite3 secara manual:

  1. Setelah berhasil, jalankan ulang Ghost:

üåê 5. Konfigurasi Domain Lokal (ghost.local)

Agar tidak mengalami redirect paksa ke HTTPS saat membuka /ghost, ikuti langkah berikut:

  1. Edit file hosts Buka dengan teks editor (sebagai Administrator):
C:\Windows\System32\drivers\etc\hosts

Tambahkan baris:

127.0.0.1    ghost.local
  1. Ubah konfigurasi Ghost Edit D:\ghost\config.development.json menjadi:
{
  "url": "http://ghost.local:2368",
  "server": {
    "port": 2368,
    "host": "127.0.0.1"
  }
}
  1. Restart Ghost
ghost restart

‚úÖ 6. Akses Ghost

  • Frontend: http://ghost.local:2368
  • Admin panel: https://ghost.deddme.id/ghost

üõ†Ô∏è 7. Skrip Shortcut (Opsional)

Buat file start-ghost.bat di D:\ghost:

@echo off
